# OS Abstraction Layer in Java

## Project Overview
This project is an Operating System Abstraction Layer built using Java for the Operating Systems course. The goal is to simulate fundamental OS functionalities, such as process management, memory management, and file system operations, providing an educational environment to understand OS concepts.
